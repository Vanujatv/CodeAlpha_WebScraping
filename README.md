# CodeAlpha Web Scraping

## Project Description
This project demonstrates web scraping using Python and BeautifulSoup. Data was collected from the Books to Scrape website and saved into a CSV file for analysis.

## Tools Used
- Python
- Requests
- BeautifulSoup
- Pandas
- Matplotlib

## Files
- WebScraping.ipynb
- scraped_data.csv
- README.md

## Output
- Successfully scraped book details from the website.
- Created a CSV dataset.
- Performed basic data analysis.
- Visualized book ratings using a bar chart.
